Getting Started

Learn how to invite FluxerSecurity to your server and set up basic protection features.

  • Invite the bot using the Invite button on the homepage.
  • Use !help to see all commands.
  • Configure roles and channels with the setup commands below.
Commands

Here are all available commands, grouped by category:

General:
  • !ping - Check if the bot is online.
  • !verify - Gain verified access to the server.
  • !help - Show this message.
Moderator Commands:
  • !kick <@user> - Kick a user from the server.
  • !ban <@user> - Ban a user from the server.
  • !timeout <@user> [minutes] - Timeout a user (defaults to 5m).
  • !warn <@user> [reason] - Issue a warning to a user (3 warns = 1hr timeout).
  • !warnings <@user> - View a user's warnings.
  • !clear <amount> - Delete up to 100 recent messages.
  • !snipe - Recover the most recently deleted message in the channel.
  • !lockdown / !unlock - Lock or unlock the current channel.
  • !slowmode <seconds> / !slowmode off - Add a specific message delay to the channel.
  • !quarantine <@user> - Completely trap all of a user's roles instantly.
  • !unquarantine <@user> - Return all of a user's roles.
  • !setlogchannel <#channel> - Set the private staff logging channel.
  • !setverifyrole <@role> - Set the role given when verifying.
  • !setquarantinerole <@role> - Set the role assigned when quarantined.
  • !antinuke on/off - Enable or disable the Anti-Nuke module.
  • !burnershield on/off - Enable or disable the Burner Account Shield.
  • !messageblocker on/off - Block all non-mod messages server-wide.
  • !whois <@user> - Investigate a user's account details.
